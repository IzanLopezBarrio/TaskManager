# TaskManager

## En que consiste:

Este programa permite al usuario crear tareas definiendo su prioridad, nombre, y otorgando una pequeña descripción de la misma.
Al crear una tarea, esta se añadirá a la lista, con un ID y la fecha de creación.

El programa ofrece la función de marcar tareas como completadas, filtrar por prioridad o estado (completado o no completado) y borrar tareas, ya sea una en concreto o la lista entera.

Adicionalmente, la lista se guarda en la memoria del equipo, por lo que el usuario no debe preocuparse de perder la lista al cerrar la ventana. Destacar que, si se borra la lista entera, esta se borrará también de la memoria.
